---
title: Vertical Conveyor
description: moves vertical between source and target position
---

![Vertical Conveyor icon](../../../assets/icons/vertical_conveyor_icon.png) 

The Vertical Conveyor transfers a loading unit from a source positon to a target position. It moves along a vertical axis and can so transfer loading units between different stories.

## Attributes
- Position (X, Y, Z)
- Orientation (°)
